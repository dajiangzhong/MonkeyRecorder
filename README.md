# Monkey Recorder

Monkey recorder from AOSP. Capable of operating Android devices on a host machine, recording
input events. Can be used in black-box testing.
Runs on Linux and OS X.

## Usage

### OS X

    cd darwin-x86/
    ./monkeyrunner monkey_recorder.py

### Linux

    cd linux-x86/
    ./monkeyrunner monkey_recorder.py
